🔗 Connect to Shared Backend

This dashboard needs a Google Apps Script URL to store shared data that everyone on your team can see. This is a one-time setup.

1. Create a new Google Sheet
2. Go to Extensions → Apps Script
3. Paste the code from google-apps-script.js (provided separately)
4. Click Deploy → New deployment → Web app
5. Set "Who has access" to Anyone
6. Copy the Web App URL and paste it below

The URL looks like: https://script.google.com/macros/s/XXXXX/exec
